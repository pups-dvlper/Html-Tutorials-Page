    <head> & <title> tag

    In HTML, the <head> element is a crucial part of the document structure and is used to provide meta-information about the web page rather than visible content.

    The <head> section typically contains metadata, links to external resources, and instructions for the browser, search engines, and other web agents.

    <title> & <link> tags

    <meta charset="UTF-8">: Specifies the character encoding for the document. UTF-8 is a widely used encoding that supports various character sets and languages.

    <title>: Defines the title of the web page, which is displayed in the browser's title bar or tab.





    <link rel="stylesheet" href="styles.css">: Links an external CSS stylesheet to style the web page's content.

    <meta>, <favicon>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">:

    Sets the viewport properties for responsive web design.

    It ensures that the page is scaled correctly on different devices and screen sizes.

    <meta name="description">: Provides a brief description of the page's content. This description is often used by search engines when displaying search results.

    <meta name="author">: Specifies the author of the web page.

    <meta name="keywords">: Provides a list of keywords related to the page's content.

    <link rel="icon" href="favicon.ico" type="image/x-icon">: Sets the website's favicon, which is the small icon displayed in the browser's tab.

    <link rel="canonical" href="canonical-url">: Specifies the canonical URL for the page, which is useful for SEO to indicate the preferred URL when there are multiple versions of a page.
